# Wildfire GIS Analysis

## Introduction
This project analyzes wildfire data using GIS and Python to identify spatial patterns and support community resilience research.

## Features
- Analyze wildfire datasets
- Perform GIS based spatial analysis
- Visualize wildfire patterns and trends
- Generate summary statistics

## Usage
1. Clone this repository.
2. Install the required Python packages.
3. Run the analysis scripts.
